## 如何练习
在 `type-challenges` 文件夹内创建对应的题目文件。<br>
然后在文件那分别创建如下两个文件：<br>
<ol>
  <li>`template.ts`：写题文件</li>
  <li>`test-cases.ts`：检测文件</li>
</ol>
也可以直接使用 vscode 的 type-challenges 插件<br>
插件做的题会保存在 .typeChallenges 中。
